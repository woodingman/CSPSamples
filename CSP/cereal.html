Cereal Challenge App

This is the chalenge final app. Play around with it a bit. What are the inputs and what are the outputs? The big difference is there are multiple user inputs and the output of the array is a list made by appending each item that meets the criteria

All the data is taken directly from this Spreadsheet

On App Lab, you will need to import this data set and create an app that filters a list into a subset of items that meets the users requirements. You must have at least two filters and use the following programming techniques

  • Function with Parameters (from user)
  • Conditionals (Selection)
  • Input (Table) and Output Arrays (Filtered List)
  • Iteration

This will culminate in you turning in Three Separate Products

  1. A link to your app with comments
  2. A written response Template
  3. A video of the app running with two different user inputs
